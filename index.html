AllEars Android App

  • Android
  • Android Studio
  • Java
  • Firebase
  • GUI
  • UX

An android app developed with a team of 3 for cs5520, mobile application development. Built using Android Studio, this app features two main modes of training: interval and chord. In interval training the user is given a random interval based on difficulty selected. In chord training the user is given a random chord quality from a pool, based on difficulty selected. Roots are randomized, and basic statistics are saved. Features the ability to set daily goals, and to check leaderboards from the Google Firebase Realtime Database associated with the app.

Sudoku Solver

  • Python3
  • Pygame
  • GUI
  • Backtracking

A Sudoku game created in Python3 with Pygame, featuring intuitive GUI alongside text and keyboard controls. The program can solve given boards, as well as generate boards that are guaranteed to have solutions. Boards are generated and solved using a Backtracking algorithm to efficiently rule out invalid partial solutions as they are deemed illegal.
